# 주제
요리학원에서 수강생의 온라인 실습 신청을 위한 시스템 입니다.  
수강생은 개설된 강좌에 수강 신청후 실습비를 입금하면 수강 신청 완료가 됩니다.   
강사는 강의를 개설하고 수강생들의 수강신청을 받습니다.  

# 개발 도구
## Spring boot
- JAVA 11
- Spring Boot Security
- Spring Data JPA
- Spring HATEOAS

## Build Tool
- Gradle

## Database
- Oracle11 xe
- H2
- Redis
- OPEN-API
